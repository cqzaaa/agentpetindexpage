AgentPet 是一款基于 Electron + React + TypeScript 架构开发的开源本地桌面 AI 助理。它常驻于您的桌面，不仅提供基于 Pixi.js 的 Live2D 虚拟体陪伴与语音互动，更能集成微信飞书等代理 24H 消息回复、动态调用 MCP 服务、四层记忆沉淀机制，办公报表定时处理等；所有聊天数据保存在本地 SQLite，数据隐私本地存储。

CORE CAPABILITIES

功能介绍：安全有度，高效代劳

AgentPet 拒绝复杂的概念包装，把最硬核的文件处理、后台托管和本地沙盒安全做到极致。

💬

微信与飞书双端托管

集成官方合规的微信 iLink SDK 与飞书网关服务。扫码后，将个人号/群聊变成 AI 答复代理。24H 后台静默流式运行，屏蔽无关信息打扰。

🛡️

命令执行安全沙盒

首创安全审批沙盒。AI 助理分析本地项目或执行命令前，强行识别拦截破坏性指令（如 rm -rf、删库等），弹出气泡授权审批，安全零隐患。

📂

多格式文档一键拖拽解析

支持 Word (mammoth/docx)、Excel (exceljs)、PDF (pdfkit/pdf-parse) 文件一键拖拽至宠物窗口。AI 本地自动提炼数据报表及要点总结。

🎭

Live2D/3D 模型热拔插

采用 PixiJS 与 ThreeJS 双核心渲染。支持一键导入外部 Live2D（`.model3.json`）及 3D 模型，支持设定语言风格，让您的桌面宠物栩栩如生。

💾

本地存储与一键数据迁移

聊天历史与技能包完全保存在本地 SQLite。支持自定义存储路径并一键迁移至您的特定盘符，数据与隐私完全掌控在您的物理硬盘中。

TECHNOLOGY STACK

技术框架：底层硬核技术架构

产品的高效与本地安全，源于底层极其强大的技术组件库。我们毫无保留地向开发者公开核心技术框架栈：

系统内核与打包

  • Electron 39.2：跨平台桌面框架封装层
  • React 19.2 + TS：前端交互界面开发
  • Vite 5.0 + Less：极速编译构建与样式管理
  • Electron-builder：一键打包 Win/Mac/Linux 客户端

桌面宠物渲染与连接

  • Pixi.js v6：高性能桌面 Live2D 宠物挂件骨骼渲染
  • Pixi-live2d-display：外部 Live2D 形象库加载解析
  • Three.js 0.184：3D 宠物形象与桌面立体场景支持
  • MCP SDK 1.29：Model Context Protocol 规范连接

多格式文档解析预览

  • ExcelJS + XLSX：Excel 电子数据表解析与写引擎
  • Mammoth + Docx-preview：Word 文档提取与本地渲染
  • PDFkit + PDF-parse：PDF 文件生成、解析与要点提炼
  • PapaParse + JSZip：CSV 处理与压缩技能包解压

持久化与微信集成

  • Better-SQLite3：高性能本地 SQL 数据库聊天持久化
  • openilink-sdk-node：微信合规官方 iLink 连接 SDK
  • ws (WebSocket)：主进程与桌面挂件进程高速双向通信
  • 命令防爆沙盒：本地自研命令过滤拦截审批机制

WECHAT INTEGRATION

集成服务：合规微信托管技术

深度解析 AgentPet 微信托管的核心机制：使用微信合规的官方 iLink 连接技术，打通大模型本地流式回复，实现高安全零封号风险的企业级部署。

🛡️ 微信集成核心技术解析

不同于市面上违规且极易被封号的网页版协议或注入式破解方案，AgentPet 采用了绝对安全的官方技术路线：

⚡

微信合规官方 iLink SDK

系统核心通过微信官方提供的 Node 平台连接器 @openilink/openilink-sdk-node 直连，所有数据通过微信专属加密管道传输，绝不篡改客户端，保障您的账号绝对安全。

💾

Better-SQLite3 本地存储与多维记忆

微信接收到的所有聊天记录、上下文关联信息及个人隐私配置均采用本地数据库持久化，坚守本地化原则，数据和控制权完全归属用户物理硬盘。

🔒

防爆指令沙盒双重过滤

微信远程执行命令支持多级指令阻断审批。若有高危指令试图远程修改或破坏您的项目，主进程安全沙盒立即将其锁定并弹窗审批，杜绝被恶意指令"删库"。
